# User Guide

## 1. Registration and Login
1. Open the application
2. Register a new account
3. Login with email and password

## 2. Tasks
1. Create a new task (title, optional description)
2. Add a deadline (optional)
3. Change status: To do / In progress / Done
4. Edit or delete tasks

## 3. Subjects (optional)
1. Create a subject (e.g., WAP, Math)
2. Assign tasks to a subject
3. Filter tasks by subject

## 4. Dashboard
- View number of tasks by status
- See overdue tasks and weekly progress
